# Allsorts Bracelet

## Summary
There are four basic parts: The Cube, The Puck, The Button and The Magnetic Lock. And then there is the Link Part, which connects the other parts. 

You can either download the STL files or create new ones using the Customizer.

Enjoy!

![photo of Allsorts bracelet](./images/allsorts1.jpg) ![photo of Allsorts bracelet](./images/allsorts2.jpg)

## Post-Printing

The Cube and the The Puck are multi-colored. I simply printed the parts separately and glued them together. 

I also glued magnets to both ends of the magnet lock parts (10 mm diameter by 2 mm Neodymium magnets). I added quite a bit of +tolerance. The attached STL file has magnetDiameter=10.5 mm, which worked well. Use the Customizer for magnets of other dimensions.

The link snaps onto the axles of the other parts by holding it at a right angle (from below). A bit of force is required.

The link was created from The Missing Link in the Customizer App using the following options:

axleDiameter = 3
wallThickness = 2
clearance = 0.3
partToPrint = 2
centerLinkWidth = 4
otherLinkWidth = 4
magnetDiameter = 10.5
magnetLength = 2
